# 🎨 Chat-Based Layout Agent

A professional web application that allows users to modify design layouts through natural language conversations with an AI assistant.

## ✨ Features

- 💬 **Natural Language Chat** - Talk to AI like a design assistant
- 🔄 **Real-time JSON Updates** - Watch layout transform instantly
- 📐 **Professional Wireframe Preview** - Visual feedback with zoom controls
- 🎯 **Aspect Ratio Conversion** - 9:16, 16:9, 1:1 support
- 📍 **Element Positioning** - Move elements to top, center, higher
- ✏️ **Text Styling** - Resize text, change colors
- 🌙 **Dark/Light Mode** - Toggle between themes
- 📁 **Layer Management** - View and select individual elements

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React 19 + Vite |
| Backend | Node.js + Express |
| AI API | Groq (Llama 3.3 70B) |
| Styling | CSS Modules + Dark Mode |
| HTTP Client | Axios |

## 📋 Prerequisites

- Node.js v18 or higher
- npm or yarn
- Groq API key (free tier available)
